# Computational Engineering with Python

Using Python to model reality — from planetary motion and optimization to machine intelligence.

---

## Who is this for?

* Engineers
* Scientists
* Students
* Software Developers
* Curious Builders

This repository is for people who want to use computation to understand and simulate the real world.
